# Multi Step Form

multi-step form 을 구현한 미니 프로젝트입니다.

## 사용 스택

다음 스택으로 구성되어있습니다.

- `react`
- `typescript`
- `next.js(pages router)`
- `emotion`
- `react-hook-form`
- `react-query`

## 시작 가이드

서비스 실행을 위해 다음을 진행합니다:

```
$ yarn install
$ yarn dev
```
